# Maintainers

This project is currently maintained by a single primary maintainer.

## Core Maintainer

- **Lead maintainer:** [@josephsenior](https://github.com/josephsenior)
- **Scope:** release management, security triage, roadmap decisions, and merge authority.

## Review and Response Targets

- **Bug reports:** initial triage within 5 business days.
- **Security reports:** acknowledged within 48 hours (see `SECURITY.md`).
- **Pull requests:** first review pass target within 7 business days.

These are best-effort targets and may vary with maintainer availability.

## Becoming a Maintainer

Potential maintainers are invited based on sustained, high-quality contributions and
demonstrated stewardship across code quality, security, and community conduct.
