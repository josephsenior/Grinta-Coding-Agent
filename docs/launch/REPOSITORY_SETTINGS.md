# Repository Discovery Settings

Apply these values in the GitHub repository settings when authenticated repository-settings access is available.

## Description

> Local-first autonomous coding agent with provider-agnostic inference, LSP, debugging, durable sessions, and a terminal UI.

## Features

- Enable Issues.
- Enable Discussions.
- Keep Releases visible on the repository front page.
- Set the website field when a stable documentation site exists.
