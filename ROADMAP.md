# Roadmap

This roadmap tracks high-level priorities for Grinta after `v1.0.0-rc1`.

## Near term

- Stabilize GA release criteria and enforce required CI/security gates.
- Improve first-run onboarding reliability on clean Windows and Linux systems.
- Expand compatibility validation for Python 3.13 and macOS.

## Mid term

- Improve long-session performance and context compaction observability.
- Expand integration/e2e coverage for provider and tooling edge cases.
- Harden release provenance and artifact transparency.

## Community requests

Open feature requests in GitHub Issues using the feature template. High-signal
requests with clear use cases are prioritized first.
